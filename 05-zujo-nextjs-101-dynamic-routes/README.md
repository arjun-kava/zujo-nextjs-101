### Setup

#### create and setup `01-zujo-nextjs-101-starter ` project

```sh
npx create-next-app 01-zujo-nextjs-101-starter --use-npm --example "https://github.com/arjun-kava/zujo-nextjs-101/tree/master/05-zujo-nextjs-101-dynamic-routes"
```

#### get into folder

```sh
cd 01-zujo-nextjs-101-starter
```

#### run command for yarn users:

```sh
yarn dev
```

#### run command for npm users:

```sh
npm run dev
```
