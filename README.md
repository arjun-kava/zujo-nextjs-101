# Zujo: Next JS tutorial for beginners codelabs

This tutorial is for next js beginners who want to start with the next js. Zujo web app has been developed completely using the next js and we had an amazing experience with the performance and ease of next js.

This tutorial includes episodes about the setup of the project, navigating between routes, adding assets, CSS & metadata, pre-rendering of page, data fetching, dynamic routes, API routes and deployment of your project
